# Evolution

This file records recurring Kronos-ORM development pitfalls and their proven fixes.

## 使用说明

- 每次构建成功后，将遇到的重要问题与解决方案记录到此文件
- 每次修复报错前,先阅读此文件了解历史踩坑情况
- 遵循已有的最佳实践,避免重复犯错

## 记录格式

```markdown
## [日期] - [问题描述]

### 问题症状
[描述问题的具体表现和错误信息]

### 问题原因
[分析问题的根本原因]

### 解决方案
[列出解决该问题的具体步骤]

### 预防措施
[列出如何避免此类问题的最佳实践]
```
