# Evolution

This file records recurring Kronos ORM usage pitfalls and proven fixes.

## 使用说明

- 修复或解释重复出现的 ORM 使用问题前，先读 `Evolution.index.md`，不要默认读取本文档全量内容。
- 只有索引命中时才用定向搜索读取本文档中的对应条目；索引未命中时不要打开全文。
- 每次记录新问题后，同步更新 `Evolution.index.md`。
- 只记录可复用、已确认的问题，不记录一次性猜测或普通功能计划。

## 记录格式

```markdown
## [日期] - [问题描述]

### 问题症状
[描述问题的具体表现和错误信息]

### 问题原因
[分析问题的根本原因]

### 解决方案
[列出解决该问题的具体步骤]

### 预防措施
[列出如何避免此类问题的最佳实践]
```
