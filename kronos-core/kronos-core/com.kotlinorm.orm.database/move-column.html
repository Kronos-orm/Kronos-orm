fun moveColumn(expect: List<Field>, current: List<Field>): List<String>

Determines the columns that need to be moved by comparing the expected and current columns.

This function identifies columns that need to be moved to match the expected order. It compares the expected columns with the current columns and returns a list of column names that need to be moved.
