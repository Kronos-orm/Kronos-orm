forObject

open override fun forObject(task: KAtomicQueryTask, kClass: KClass<*>, isKPojo: Boolean, superTypes: List<String>): Any?

Executes a query task and returns the result as a single object.

Return

An object of the specified type, or null if the result set is empty. If isKPojo is true, the object is created as a KPojo instance. Otherwise, it is created as a plain object.

Parameters

task

The query task to be executed.

kClass

The Kotlin class of the target object type.

isKPojo

A flag indicating whether the target type is a KPojo.

superTypes

A list of super types for the target object type.

Throws

UnsupportedTypeException

If the target type is a KPojo and its fields cannot be found.
